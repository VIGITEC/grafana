La plataforma de código abierto para monitoreo y observabilidad

[![Licencia](https://img.shields.io/github/license/grafana/grafana)](LICENSE)
[![Informe de Go](https://goreportcard.com/badge/github.com/grafana/grafana)](https://goreportcard.com/report/github.com/grafana/grafana)

Grafana te permite consultar, visualizar, alertar y entender tus métricas sin importar dónde estén almacenadas. Crea, explora y comparte paneles con tu equipo y fomenta una cultura basada en datos:

- **Visualizaciones:** Gráficos rápidos y flexibles del lado del cliente con múltiples opciones. Los plugins de panel ofrecen muchas formas diferentes de visualizar métricas y registros.
- **Paneles dinámicos:** Crea paneles dinámicos y reutilizables con variables de plantilla que aparecen como menús desplegables en la parte superior del panel.
- **Explorar métricas:** Explora tus datos a través de consultas ad-hoc y exploración dinámica. Vista dividida y compara diferentes rangos de tiempo, consultas y fuentes de datos una al lado de la otra.
- **Explorar registros:** Experimenta la magia de cambiar de métricas a registros con filtros de etiquetas preservados. Busca rápidamente en todos tus registros o transmítelos en vivo.
- **Alertas:** Define visualmente reglas de alerta para tus métricas más importantes. Grafana evaluará continuamente y enviará notificaciones a sistemas como Slack, PagerDuty, VictorOps, OpsGenie.
- **Fuentes de datos mixtas:** ¡Mezcla diferentes fuentes de datos en el mismo gráfico! Puedes especificar una fuente de datos por consulta. Esto funciona incluso para fuentes de datos personalizadas.
